# 🍜 Solusi Mitra UMKM - Konsultan Bisnis F&B UMKM Indonesia

Dashboard analisis tren kuliner berbasis **Google Trends** untuk membantu pelaku UMKM Kuliner Indonesia mengidentifikasi tren makanan & minuman, peluang menu, dan pergeseran musiman.

## ✨ Fitur Utama

- 📊 **Grafik Minat Penelusuran** — Visualisasi tren pencarian kuliner secara real-time
- 🗺️ **Analisis Regional** — Identifikasi provinsi dengan minat tertinggi
- 🔍 **Kueri Terkait** — Temukan kueri terpopuler dan naik daun (breakout)
- 🤖 **Konsultan AI** — 4 pilar rekomendasi bisnis:
  - 🏆 **Aksi Cepat** — Langkah taktis berdasarkan status tren
  - 💼 **Strategi Bisnis** — Menu engineering, harga, ekspansi
  - 🎬 **Ide Konten** — Hook viral TikTok/Reels, challenge, storytelling
  - 📣 **Digital Marketing** — GoFood SEO, IG Ads, WA referral, hashtag

## 🚀 Cara Menjalankan

```bash
# Install dependencies
pip install -r requirements.txt

# Jalankan server
python app.py
```

Buka browser di `http://127.0.0.1:5000`

## 🛠️ Tech Stack

- **Backend:** Python Flask + PyTrends
- **Frontend:** HTML5, CSS3 (Dark Theme), JavaScript, Chart.js
- **Icons:** Lucide Icons
- **Fonts:** Plus Jakarta Sans (Google Fonts)
